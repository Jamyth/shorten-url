# **URL Shortener**

A simple web application for shortening long url into short.

[DEMO](https://url.jcstudio.com.hk/)

## **Tech Stack:**

Backend: `NestJS`

Frontend: `React`

## **Featuring Library (self-made):**

- jamyth-web-util
- nest-api-generator
- api-code-generator
- eslint-config-iamyth
- @iamyth/prettier-config
- @iamyth/webpack-runner
